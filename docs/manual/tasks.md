# Task Board

**Route:** `/events/:id/tasks`

---

## The Big Picture

Running an event means tracking dozens of things to do:
- "Call the caterer"
- "Confirm the playlist"
- "Test the sound system"

The Task Board helps you organise all of this like sticky notes on a wall.

---

## The Board

Tasks are arranged in **columns** like a kanban board:

| Column | What it means |
|--------|---------------|
| **Pending** | Not started yet |
| **In Progress** | Someone is working on it |
| **Done** | Completed |
| **Blocked** | Can't move forward (something is stopping it) |

### Drag & Drop
Click and drag a task card from one column to another to update its status.

---

## Creating a Task

1. Click **Add Task** in any column
2. Fill in:
   - **Title** (required) — what needs to be done
   - **Description** (optional) — details, notes, instructions
   - **Assignee** (optional) — who's responsible
   - **Due date** (optional) — when it needs to be done
   - **Priority** — low, medium, high, critical
3. Click **Save**

The task appears in the **Pending** column.

---

## Task Details

Click on any task card to see full details:
- **Comments** — add notes or updates
- **Attachments** — upload files
- **Status history** — when it moved between columns

---

## Overdue Tasks

Tasks past their due date show up:
- **Red highlight** on the task card
- Listed in the **Upcoming Deadlines** section on the Event Details page

---

## What Coordinators See

Coordinators can:
- View all tasks
- Drag tasks between columns
- Add comments
- **Cannot** create, edit, or delete tasks

---

## Pro Tips
- Set **due dates** so overdue tasks show up on the Event Dashboard
- **Assign tasks** to specific people so everyone knows their responsibility
- Use **priorities** to focus on what matters most
- Tasks with due dates appear on the Event Details **Upcoming Deadlines** widget
